# assets

Сюда положите свои фотографии услуг и фон главного экрана.

Рекомендации:
- Формат: `.webp` или `.jpg`, оптимизированные (до ~300 КБ).
- Фото услуг: пропорции ~16:10, минимум 800×500 px.
- После добавления замените блоки `.service-media` в `index.html` на `<img>`,
  а фон в `styles.css` (`.hero-bg`).
